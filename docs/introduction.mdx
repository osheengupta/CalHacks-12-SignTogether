# Welcome to SignTogether

SignTogether is a real-time, accessible video-meeting platform that uses computer vision and AI to make communication between deaf and hearing users seamless.

## What is SignTogether?

SignTogether bridges the communication gap in video meetings by providing:

- **Real-time video meetings** powered by LiveKit
- **Live speech-to-text captions** using Deepgram
- **AI-powered sign language detection** with Gemini Vision
- **Intelligent meeting summaries** generated by Claude 3.5 Sonnet
- **Gesture recognition** for common signs like "yes", "no", "thank you"
